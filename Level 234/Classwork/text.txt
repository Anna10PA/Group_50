pipenv - არის ვირტუალური გარემო. გვაძლევს იმის საშუალებას რომ გავუშვათ დამატებითი ქომანდები.გამოიყენება პითინისთვის. აცალკევებს პროექტს.(მაგალითად ერთგან ჯანგოს კოდი რომ დაიწეროს და მეორე ტერმინალში პითონის). სჭირდება დაინსტალირება. და არის ხელსაწყო. რომ დავაკავშიროთ vite-ს ჰგავს. ჩავთვალოთ კონკრეტული ერთი პროექტი. 

pip install pipenv - // აინსტალირებს pipenv-ს pip-დან.

pipenv shell - // pipenv-ის ტერმინალს ვხსნით

pip install django - გლობალურად, თვითონ კომპიუტერში აინსტალირებს ჯანგოს.
pip install django==4.0 თუ არ არის დაინსტალირებული მაშინ ამ ვერსიით დაინსტალირებს. თუ არის უკვე მაშინ ამ ვერსიით შეცვლის

pipenv install django - თვითონ ერთ კონკრეტულ პროექტში აინსტალირებს ჯანგოს. 
python -m django --version (ვნახულობთ ვერსიას)

django-admin --version - // ვნახულობთ ჯანგო ადმინის ვერსიას

django-admin startproject Folder_name - // startproject ქმნის ახალ პროექტს და მის შემდეგ ვწერთ ფოლდერს რაც გვინდა რომ დავარქვათ (პროექტის სახელი). მხოლოდ django არ გვაქვს.
