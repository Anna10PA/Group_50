RGB

rgb ნიშნავს red, green, blue.
სწორედ ამ თანმიმდევრობით არის შესაძლებელი ფერების შერევა. თუ რამესთვის ფერის მინიჭება გვინდა ჩავწერთ რომელი ფერი რამდენით გვინდა. მინიმალური 0 (მუქი) არის, მაქსიმალური (მკვეთრი) 255.

255, 0, 0 - წითელი (მკვეთრი წითელი),
0, 255, 0 - მწვანე (მკვეთრი),
0, 0, 255 - ლურჯი(მკვეთრი),
0, 0, 0 - შავი (ყველაზე მუქი რამ რაც შეიძლება იყოს), უფერო
255, 255, 255 - თეთრი (ყველაზე ღია რამ რაც შეიძლება იყოს)

დადებითი მხარე:
1) მარტივი გამოსაყენებელია და ექსპერიმენტობას შეიძლება, (რედაქტირება, შეცვლა)
2) აქვს გამჭირვალობის პარამეტრიც (rgba)
3) მარტივად იკითხება


ფერები rgb:

color:rgb(255, 255, 255)

color:rgb(0, 0, 0)

color:rgb(255, 0, 0)

color:rgb(0, 255, 0)

color:rgb(0, 0, 255)

color:rgb(255, 255, 0)

color:rgb(255, 0, 255)

color:rgb(0, 255, 255)

Hex

hex შედგება 6 სიმბოლოსგან რომელსაც წინ აქვს # ნიშანი. და სამ წყვილად რომ დავყოთ იგივეა რაც rgb.
პირველი ორი სიმბოლო არის წითელი, მეორე წყვლი არის მწვანე, მესამე ლურჯი. შეიძლება ვთქვათ რომ 16 მნიშვნელი აქვს. 0-9-მდე და მე-10-16 მაგივრად A-F არის.
FF = 255 (max)
00 = 0 (min)
#ffffff - თეთრი (შემოკლებით #fff)
#000000 - შავი (შემოკლებით #000)
